# Recetas

App de recetas para guardar, consultar y compartir las recetas que vamos encontrando en TikTok, Instagram y otros sitios.

## Idea

Una PWA (Progressive Web App) sencilla para dos personas. Se abre desde el móvil como una app normal, funciona offline, y las recetas se sincronizan en tiempo real entre los dos.

## Funcionalidades previstas

- Añadir recetas (título, ingredientes, pasos, link al vídeo original)
- Ver todas las recetas guardadas
- Buscar por nombre o ingrediente
- Funciona offline (útil en la cocina)
- Se instala en el móvil desde el navegador (sin app store)
- Sincronización en tiempo real entre usuarios

## Stack

- **Frontend**: React + Vite (PWA)
- **Backend/DB**: Supabase (gratis)
- **Hosting**: Vercel (gratis)

## Coste

Cero. Todo el stack es gratuito para este nivel de uso.
